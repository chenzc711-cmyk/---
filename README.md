# 拾纷电商管理系统（微信小程序）

一个面向天猫店铺风格的微信小程序仓库管理系统原型，包含公司代码登录、仪表盘、库存管理、入库、退货入库、出库与成本登记。

## 登录

进入小程序后需要输入公司代码，默认公司代码为：`sfds`。

## 功能模块

- 首页仪表盘：展示商品数量、库存数量、库存总金额、库存预警、供应商货款余额、今日出入库数量与金额、本月入库/出库累计数据与金额。
- 库存管理：维护商品分类、商品名称、规格、单价、在库数量、供应商、备注、预警数量。
- 入库管理：登记入库日期、商品名称、规格、单价、数量、供应商、备注，并支持按日期筛选。
- 退货入库管理：登记退货日期、订单号、物流公司、物流单号、商品名称、规格、数量、货物状态、备注，并支持按日期筛选。
- 出库管理：登记出库日期、商品名称、规格、数量、单价、总金额、用途/备注，并支持按日期筛选。
- 成本登记：登记支出时间、支出金额、支出用途、支出分类、备注，并支持按日期筛选。

> 注意：微信小程序原生 `tabBar.list` 最多支持 5 项，因此底部 Tab 保留「仪表盘、库存、入库、退货、出库」5 项；「成本登记」仍是完整独立页面，可从仪表盘的「成本登记入口」进入。

所有登记页面均支持记录编辑、全选和一键删除所选记录。数据默认保存在微信小程序本地缓存中，便于快速演示和二次开发。

## 运行方式

使用微信开发者工具导入本项目根目录，选择小程序模式运行即可。
